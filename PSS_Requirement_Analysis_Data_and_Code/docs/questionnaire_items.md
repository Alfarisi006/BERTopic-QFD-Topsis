# Questionnaire Items

This file provides the open-ended questionnaire items used to collect user requirement texts for the campus shared wardrobe service case study.

## Research context

The questionnaire was designed to collect potential users' experiences, expectations, functional needs, and adoption concerns regarding a campus shared wardrobe service.

In the case study, 101 valid questionnaires were obtained. The questionnaire contained four open-ended questions. Each response to each question was treated as an independent unit of analysis, resulting in 404 open-ended responses. After data cleaning, 397 valid texts were retained for BERTopic-based requirement theme extraction.

## Open-ended questionnaire items / 开放式问卷题项

### Q1. Wardrobe-related frustrations / 衣物管理相关困扰

**Chinese original / 中文题项：**  
你在校园生活中的衣物收纳、衣物管理或换季整理过程中遇到过哪些困难或不便？

**English translation / 英文翻译：**  
What difficulties or inconveniences have you experienced in clothing storage, wardrobe management, or seasonal clothing organisation during campus life?

---

### Q2. Ideal shared wardrobe experience / 理想共享衣橱体验

**Chinese original / 中文题项：**  
如果校园中提供共享衣橱服务，你理想中的使用体验是什么样的？

**English translation / 英文翻译：**  
If a campus shared wardrobe service were provided, what would your ideal user experience look like?

---

### Q3. Expected system functions / 期望的系统功能

**Chinese original / 中文题项：**  
你希望校园共享衣橱系统具备哪些功能或服务？

**English translation / 英文翻译：**  
What functions or services would you expect from a campus shared wardrobe system?

---

### Q4. Major adoption concerns / 主要使用顾虑

**Chinese original / 中文题项：**  
哪些问题或顾虑会影响你使用校园共享衣橱服务的意愿？

**English translation / 英文翻译：**  
What issues or concerns would affect your willingness to use a campus shared wardrobe service?

## Data processing note

The responses were anonymized before publication. Blank responses, duplicated responses, and responses unrelated to the questionnaire context were removed during data cleaning. Expressions related to usage experience, emotional perception, and value judgment were retained to preserve information relevant to user-perceived value.
